# Aprendiendo Dart para el desarrollo logico

Este proyecto tiene como objetivo principal guiar a los nuevos desarrolladores en el aprendizaje de Dart, enfocandose en la logica de programacion. Contiene ejercicios y ejemplos de codigo que se pueden utilizar para aprender los conceptos basicos de Dart.

Esta estructurado de la siguiente manera:

* La carpeta `src` contiene los ejercicios y ejemplos de codigo.

Cada ejercicio y ejemplo esta  diseñado para que el desarrollador pueda probar y aprender diferentes conceptos de Dart.
